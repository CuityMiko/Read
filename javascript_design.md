# 第七章 设计模式类别

## 1. 创建型设计模式

1. Constructor(构造器)
2. Factory(工厂)
3. Abstract(抽象)
4. Prototype(原型)
5. Singleton(单例)
6. Builder(生成器)

## 2. 结构型设计模式

1. Decorator(装饰器)
2. Facade(外观)
3. Flyweight(享元)
4. Adapter(适配器)
5. Proxy(代理)

## 3. 行为设计模式

1. Iterator(迭代器)
2. Mediator(中介者)
3. ObSERVER(观察者)和Visitor(访问者)
